总览

职业道德

会计人员：

  1. 坚持诚信，守法奉公。
  2. 坚持准则，守则敬业。
  3. 坚持学习，守正创新。

中国注册会计师职业道德：

  1. 诚信。
  2. 客观公正。
  3. 独立性：鉴证业务的灵魂。
  4. 专业胜任能力和勤勉尽责。
  5. 保密。
  6. 良好职业行为。

职业职能

会计是以货币为主要计量单位，运用专门的方法，核算和监督一个单位经济活动的一种经济管理工作。

会计核算与会计监督，两者相辅相成，会计核算职能是监督职能的基础，监督职能是核算职能的保障。

会计基本假设

做账时所需的合理设定。

假设 说明
会计主体（空间范围） 给谁做账、谁就是会计主体。法律主体（注册公司等）必然是会计主体，但会计主体不一定是法律主体。
持续经营（时间范围） 会计主体在可预见的将来会持续正常经营，不会停业，不会大规模削减业务。是会计分期的前提，若企业进入破产阶段应考虑破产假设。
会计分期 做账要分期，进而出现折旧、摊销等会计处理方法。
货币计量 以货币为主要计量单位去反映会计主体的生产经营活动。

会计基础

企业会计的确认、计量和报告应当以权责发生制为基础。

制度 权责发生制 收付实现制
说明 以权利、责任发生的时间为准进行会计确认的制度 以现金的收到或支付为准进行会计确认的制度
适用行业 企业会计、行政事业单位的财务会计 行政事业单位的预算会计
适用报表 资产负债表、利润表、所有者权益变动表 现金流量表

老板分别聘请张三李四各担任CEO一个月，张三李四都赚了1000w，成本800w，但李四经营时涉及了房租要债200w，利润归零。

老板认为张三能力更强，则老板是站在收付实现制角度思考。

李四不服气，他认为张三经营时也使用了场地，也应分摊租地成本，各100w，则李四是站在权责发生制角度思考。

⭐会计信息质量要求

质量要求 定义理解 应用
可靠性 真实完整、中立无偏。 -
相关性 有用性、有价值性。 -
可理解性 清晰明了，便于理解。 -
可比性 纵向可比，同一企业不同时期可比；
横向可比，不同企业同一时期可比。		 有统一标准才能进行对比，会计做账准则不得随意变更。
实质重于形式 经济实质重于法律形式。 售后回购，回购价可确定时不确认收入。
出售附有追索权的商业汇票。
合并范围的确定、合并抵销分录的编制。
反向购买。
金融负债和权益工具的区分。
重要性 从财务报告使用者的角度判断。 不重要的前期差错可不进行追溯但必须调整。
实际利率和票面利率相差不大时，可用票面利率计算投资收益。
谨慎性 不高估资产或收入，不低估负债或费用。（不吹牛） 对资产计提减值准备。
或有负债满足预计负债的确认条件时应确认预计负债。
或有资产只有在实际发生时才能予以确认。
企业不得设置秘密准备。
及时性 不得提前或延后。 -

任何不符合会计准则规定的处理，均属于滥用会计政策。

会计要素

确认

会计要素 理解 注意事项
资产 将要购买的商品、盘亏的商品、筹建期间发生的开办费用、为广告营销等特定活动采购的商品等不属于资产
负债 现时义务可确认为负债 潜在义务不能确认为负债
所有者权益 资产 - 负债 = 所有者权益 其他综合收益：直接计入所有者权益的利得和损失。
资本公积 - 其他：股份支付权益结算、长投权益法。
收入 日常活动（利得为非日常） 收入：主营业务收入、其他业务收入。
利得：直接计入当期损益的利得（营业外收入）和直接计入所有者权益的利得（其他综合效益）。
费用 日常活动（损失为非日常） 费用：主营业务成本、其他业务成本、管理费用、销售费用、财务费用、税金及附加、资产减值损失等。
损失：直接计入当期损益的损失（营业外支出）和直接计入所有者权益的损失（其他综合收益）。
利润 利润 = ( 收入 - 费用 ) + ( 利得 - 损失 ) 营业利润 = 营业收入 - 营业成本 - 税金及附加 - 管理费用 - 销售费用 - 财务费用 - 资产减值损失 ± 公允价值变动损益 ± 投资收益
利润总额 = 营业利润 + 营业外收入 - 营业外支出
净利润 = 利润总额 - 所得税费用

资产的“账面价值”与“账面余额”。

账面价值是指账面余额减去相关备抵项目后的净额；账面余额是指该账户的账面实际余额，不扣除备抵项目的金额。

资产 解释
固定资产
无形资产
投资性房地产（成本模式）		 账面余额 = 账面原值
账面净值 = 原值- 累计折旧/摊销
账面价值 = 原值- 累计折旧/摊销 - 减值准备
长期应收款 账面余额 = 一级科目的余额
账面价值 = 一级科目的余额 - 未实现融资收益 - 坏账准备

计量

会计要素 理解 注意事项
历史成本 购买设备时支付10w元 适用于大部分情况
重置成本 现在再购买相同的设备，只需6w元 主要针对盘盈的资产
可变现净值 估计售价 - 加工成本 - 估计销售税费 主要针对期末存货
现值 对未来现金流量折现后的价值 主要针对资产减值
公允价值 脱手价格 -

财务报告（财报）

目标

决策有用观：向财务报告使用者提供与企业财务状况、经营成果和现金流量等有关的会计信息；有助于财务会计报告使用者作出经济决策。

受托责任观：反映企业管理层受托责任履行情况。

组成部分

四表一注。

  1. 资产负债表
    • 反映企业财务状况的时点表。
    • 结构为账户式。
  2. 利润表
    • 反映企业经营成果的时期表。
    • 结构为多步式。
  3. 现金流量表
    • 反映企业现金及现金等价物流入流出情况的时期表。
  4. 所有者权益变动（或股东权益）表
  5. 附注

存货

存货，是指企业在日常活动中持有以备出售的产成品或商品、处在生产过程中的在产品、在生产过程或提供劳务过程中耗用的材料、物料等。

存货 非存货
原材料、在产品、半成品、产成品、商品、周转材料；
委托加工物资是委托方的存货；
委托代销商品是委托方的存货；
企业接受外来原材料加工的代制品、为外单位加工修理的代修品，制造和修理完成验收入库后，加工成本和修理成本形成企业存货；
房地产开发企业用于建造商品房的土地使用权及商品房属于企业的存货（开发产品）；
已经取得商品所有权，但尚未验收入库的在途物资；
已发货但存货的风险和报酬并未转移给购买方的发出商品。		 企业采购用于广告营销活动的特定商品，不能为其带来经济利益流入，不属于存货，计入销售费用；
盘亏、损毁的材料等，不能为企业带来经济利益流入，不属于存货；
受托代销商品不属于受托方的存货，填列资产负债表“存货”项目时“受托代销商品”与“受托代销商品款”两科目一增一减相互抵销，不列为受托方存货；
为建造固定资产等各项工程而储备的各种材料，属于工程物资。

存货确认

  1. 与该存货有关的经济利益很可能流入企业。
  2. 该存货的成本能够可靠地计量。

初始计量

企业取得存货应当按照成本计价。

存货成本（价、税、费）：

  1. 采购成本
  2. 加工成本
    • 车间固定资产修理费
    • 季节性和修理期间停工损失
  3. 到达目前场所和状态所发生的其他成本

以下不计入成本：

项目 处理原则
非合理损耗 计入管理费用或营业外支出等
但合理损耗计入成本
入库后的仓储费用 计入管理费用
但生产环节所必须的仓储费计入成本
同时酒水等商品因陈酿需要必须产生的仓储费计入成本
用于广告营销活动费用 计入销售费用
商品流通企业进货费用 进货费用金额较小的，可直接计入销售费用
